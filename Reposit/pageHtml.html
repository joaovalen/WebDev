Alguns Elementos Importantes
Formatação Automática VsCode

No Vs Code você pode acessar a paleta de comandos com ctrl + shift + p, e pesquisar pelo comando format document

Texto (tags <P>, <BR>,.…)

<p> se refere a parágrafos de texto, enquanto <br> pode ser usado para quebrar as linhas:

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Fugiat omnis, unde corrupti libero <br>
dicta architecto praesentium possimus. Adipisci cumque corrupti iste officia, eum animi dolores blanditiis dolorum! Facilis, nostrum blanditiis?</p>

Anchor Tags - Links para outras páginas (tag <a>,...)

Com a tag de link, podemos levar o usuário a outras páginas na internet ou até mesmo para outras seções de nossa própria página:

<a>href="https://google.com" target="_blank">Google</a>

Google

<a>href="https://twitter.com" target="_blank">Google</a>

Twitter

É importante destacar a propriedade "target", que ao ser setada para "_blank" abre uma nova guia ao clicar.

Imagem (tag <figure>, <img>, <figurecaption>)

Podemos usar a tag <img> para inserir imagens

Ao usar a característica alt="" estamos descrevendo para leitores de tela

<img href="assets/cat.png" alt="a cat on the wind">

Lista (tags <ul>,<li>,<ol>,...)

Podemos criar listas não ordenadas <ul> e listas ordenadas <ol>:

Uma lista sem ordem

  • Clonar cartão
  • Desviar dinheiro
  • Comprar pão

Uma lista com ordem

  1. Ir correr
  2. Dar comida pro cachorro
  3. Tomar banho

Semantic Elements

Ao invés de usar sempre Divs, varie com:
<section> Seções mesmo, vc que sabe como usar, partes diferentes do conteúdo etc
<article> Digamos algo independente e reutilizável
<nav> Tudo que forneça links de navegação para outras páginas etc
<main> Conteúdo principal, excluindo navbars, sidebars, etc
<header> é o header ne
<footer> é o footer neam
<aside> Algo não essencial, sidebars e etc
<summary>
<details>

Forms

O elemento <form> por si só é só um container sem impacto visual.
Adicionamos dentro deles uma coleção de inputs, checkboxes e etc.
representa uma seção de um documento contendo controles interativos para enviar informação
